Adverse Yaw — 조종 입력이 만드는 문제
조종사가 오른쪽 뱅크를 위해 Aileron을 조작하는 순간, 왼쪽 날개 양력↑ = 유도항력↑ → 왼쪽 날개가 뒤로 당겨짐 → 기수가 의도와 반대인 왼쪽으로 Yaw. 조종사는 선회 방향(오른쪽)으로 Rudder를 밟아 상쇄해야 합니다.
성격: 조종성 원인: 유도항력 차 주기: 없음 대응: Rudder · Frise · Differential
Dutch Roll — 스스로 일어나는 진동
돌풍 같은 외란으로 β(옆미끄럼각) 발생 → Vertical tail이 복원 Yaw → 관성으로 오버슛 → 반대 β → 상반각·후퇴각이 Roll 유발 → 다시 β 생성 → Roll과 Yaw가 맞물려 3~8초 주기로 지속 진동. Roll stability > Yaw stability일 때 발생.
성격: 동안정성 원인: Roll-Yaw 결합 주기: 3~8초 대응: Yaw Damper

Adverse Yaw

Dutch Roll
외란 후 자연 발생 · Roll–Yaw 8자 궤적 · 감쇠될 때까지 반복
